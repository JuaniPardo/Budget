Budget Online

Bienvenido a Budget Online, un proyecto de Juan Pardo para el curso de JavaScript en CoderHouse.

Este proyecto es un ejemplo de cómo se puede crear un proyecto web de finanzas usando HTML, CSS y JavaScript. Este proyecto se ha creado para ser una introducción al mundo de JavaScript y se ha optado por usar Tailwind CSS para darle un aspecto moderno y atractivo.

Si tienes alguna pregunta o necesitas ayuda, no dudes en contactarnos a través de mi repositorio de GitHub.
